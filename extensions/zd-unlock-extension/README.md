# CS Bot Unlock Extension

Chrome 扩展后台轮询 `csbot` 的解锁命令，并从白名单网络直接请求后台解锁接口。

## 安装

1. 打开 Chrome `chrome://extensions/`
2. 开启右上角 `Developer mode`
3. 点击 `Load unpacked`
4. 选择本目录：`extensions/zd-unlock-extension`
5. 点击扩展图标，确认状态为启用

## 使用

扩展安装后只要 Chrome 运行，就会每 30 秒唤醒一次，并使用长轮询等待命令。

弹窗里保留后台登录状态和接口录制器。轮询固定开启，接口地址、命令密钥和解锁参数由扩展内部使用，不在弹窗展示。

当前支持后台动作：

- 短信/验证码限制
- 登录密码试错限制
- 代理 IP 加白
- 迁移米兰（使用 6站点后台，需先打开 6site 页面）
- 场馆注单查询（使用场馆后台，需先打开 merchant-own-backstage 页面）

## 接口录制器

用于录制一整段后台手动操作产生的 Fetch/XHR 接口，方便后续做自动流程。

1. 打开已登录的后台页面（支持 9site、6site 和场馆后台）
2. 点击扩展图标
3. 点击 `开始录制`
4. 手动完成要自动化的操作流程
5. 点击 `停止录制`
6. 点击 `导出 JSON`

录制内容包括：

- 请求时间和顺序
- 请求 URL、method、body
- 返回状态码和返回内容
- 页面地址

导出的 JSON 可直接发给开发者分析接口流程。敏感请求头会自动标记为 `[redacted]`，实际自动执行仍会使用扩展从页面捕获到的当前登录态。

## 测试

1. 群内发送：`Wesley333 短信解锁`
2. 打开扩展弹窗查看状态
3. 成功时显示对应动作的 `HTTP 200`

如果状态是 `轮询失败`，说明扩展访问不了 csbot。
如果状态是 `解锁失败` 或非 200，说明请求头、登录态或白名单网络有问题。
